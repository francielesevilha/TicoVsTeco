# Tico VS Teco

O Tico VS Teco é um jogo da velha criado com JavaScript, PHP, HTML e CSS para a disciplina de Padrões de Projetos de Sistemas da Fatec de Bauru/São Paulo




## 🛠 Tecnologias utilizadas
Javascript, PHP, CSS, HTML
